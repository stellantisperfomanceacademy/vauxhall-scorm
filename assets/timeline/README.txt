TIMELINE BACKGROUNDS
====================

Drop one JPG per timeline year into this folder.

Required filenames (match the year in content.js):
  1213.jpg   — name origin / Falkes de Breauté
  1857.jpg
  1903.jpg
  1904.jpg
  ...

Until you add a JPG, the author tool uses the matching .svg placeholder
(e.g. 1213.svg). Landscape 960×540 or wider works best.
